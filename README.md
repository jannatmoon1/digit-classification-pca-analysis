# digit-classification-pca-analysis
A machine learning pipeline that classifies handwritten digits using Logistic Regression and evaluates how different PCA dimensions affect model accuracy.
## 🛠️ Required Libraries
Before running the notebook, ensure you have the following Python libraries installed:
* **NumPy:** For handling large mathematical arrays.
* **Pandas:** For structuring and analyzing data.
* **Scikit-Learn:** For loading the digits dataset, running PCA, and training the Logistic Regression model.
## 🏃‍♂️ How to Run This Project
Follow these simple steps to run the notebook directly in your browser using Google Colab:

1. **Open the Notebook:** Click the "Open in Colab" badge at the top of the notebook file inside this repository.
2. **Connect to a Runtime:** Click the **Connect** button in the top right corner of the Colab window.
3. **Install Missing Libraries (If Any):** If you run this locally or on a fresh environment, you can install the required packages by running:
   ```bash
   pip install numpy pandas scikit-learn matplotlib seaborn
   ```
4. **Run the Code:** 
   * To run cells one by one, click the **Play** button on the left of each cell.
   * To run everything at once, click **Runtime** in the top menu and select **Run all** (or press `Ctrl + F9`).
5. **View Results:** Scroll down to the bottom of the notebook to view the performance graphs and accuracy scores.
